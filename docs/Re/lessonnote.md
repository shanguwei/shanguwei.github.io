# IDA快捷键

界面转换 -空格

字符串 - shift + f12

调用关系- x

反编译- f5

**注释 - /**

**重命名 - n**

改变数值格式 - 右键或者r

提取数据 - shift + e

# 逆向做题思路

1、执行程序，掌握程序执行逻辑

2、shift + f12 找到关键字符串，双击找到地址

3、找到是谁调用了这个字符串，点击变量名，按x查看调用关系

4、找到关键函数后，f5反编译，阅读伪代码

5、根据伪代码及调试，确定程序逻辑和flag验证机制

6、分析程序如何才能满足校验，根据校验顺序逆推逻辑

分析结果：

flag的每一位与key的每一位异或，结果就是result

7、整合已知数据，判断已知数据与flag的关系，编写脚本解题

1100 flag

1011 key

0111 result



1011 key

0111 result

1100 flag
